## ObjectDock 是什么？

ObjectDock 是由美国老牌 Windows 定制软件公司 **Stardock** 出品的 Dock 栏工具，在 Windows 上运行，效果跟 macOS 的 Dock 几乎一模一样——图标放大动画、悬停缩放、自动隐藏，一个都不少。

它不只是个「好看的快捷方式」，还能**完全替代 Windows 原生任务栏**，或者和任务栏并存。你可以把它放在桌面任意边缘，上下左右随你选。

最新版本 ObjectDock 3 是一次彻底重写，从 32 位升级为 **64 位架构**，原生支持 Windows 10 和 Windows 11，还加入了高分辨率显示屏适配（HiDPI）、UWP 应用程序固定支持，以及更流畅的缩放动画效果。对于那些用了旧版多年的用户来说，这次更新可以说是等了很久的一次「真正的进化」。

## 两种 Dock 模式，各有妙用

ObjectDock 支持两种完全不同的 Dock 类型，可以单独用，也可以同时在桌面上跑多个：

**程序坞（Quick Launch Dock）**
就是经典的 Mac Dock 样式。把常用软件拖进去，鼠标移过去图标就会放大弹起来，点击直接启动。可以设置成「替代任务栏」模式，把原生任务栏隐藏掉，桌面顿时清爽一大截。

**标签卡 Dock（Tabbed Dock）**
这个模式稍微小众一点，但非常实用。可以把一堆文档、链接、工程文件夹按类型分门别类放进去，类似文件管理器侧边栏的感觉。比如一个标签放「工作文件」，一个放「常用网站」，不需要的时候缩到屏幕边缘，需要的时候鼠标一靠就滑出来。

这两种模式都可以自由定制：图标大小、背景颜色、透明度、磨砂效果、字体……甚至可以从社区下载主题皮肤来换风格。

## 主要功能亮点

- **动画效果**：图标支持缩放、晃动、闪烁等多种悬停动画，跟 macOS 的弹跳效果非常接近
- **多 Dock 支持**：可以同时开多个 Dock 栏，分别放在屏幕不同位置
- **自动隐藏**：不用时自动收起，鼠标移过去才显示，最大化桌面空间
- **置顶模式**：设置 Dock 始终在其他窗口上方显示，随时可访问
- **系统任务栏替换**：可以完全隐藏 Windows 原生任务栏，只用 ObjectDock
- **高分屏支持**：4K 显示器下图标依然清晰锐利，不虚化
- **UWP 应用固定**：Windows 商店的应用也能添加到 Dock 里了（旧版做不到）
